# Generated assets

All visuals (characters, vehicles, icons, sounds) are original — drawn as SVG/CSS or generated in-house via Phaser Graphics / Canvas 2D. Do not add copyrighted art, logos, character likenesses, or audio from existing franchises (Cars, Pixar, Disney, Marvel, etc.) to this directory or anywhere in the project.

Cars are drawn as a clear side-view profile (body, windowed cabin, two big wheels) — see `game/systems/carTextures.ts` — rather than a top-down racing-game sprite, since a side profile is the silhouette a young child actually recognizes as "a car."
