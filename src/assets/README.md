# Generated and licensed assets

Most visuals (icons, environment, UI shapes) are original — drawn as SVG/CSS or generated in-house via Phaser Graphics. Do not add copyrighted art, logos, character likenesses, or audio from existing franchises (Cars, Pixar, Disney, Marvel, etc.) to this directory or anywhere in the project.

## Third-party assets in use

- **Car sprites** (`public/assets/cars/car_red.png`, `car_blue.png`, `car_green.png`, `car_black.png`) — from Kenney's "Racing Pack" (https://kenney.nl/assets/racing-pack), licensed CC0 (public domain, no attribution required, free for commercial use): https://creativecommons.org/publicdomain/zero/1.0/. Renamed from `car_red_1.png` etc. A drawn face (eyes + smile) is composited on top at runtime so the cars keep Kinder Quest's original "talking car" personality — see `game/systems/carTextures.ts`.
